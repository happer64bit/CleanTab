# Build

## Windows
```powershell
cd CleanTab

npm install

build-windows

```
## Linux, Mac, Unix
```bash
cd CleanTab

npm install

make build
make fix-file-size # For deleting *.map files
```

## Building Zip File
```bash
# Windows
py.exe scripts/build-zip.py

# Linux, Mac, Unix
python scripts/build-zip.py
```
